Titre

Pour aller d'une page à une autre sur un site web, il faut utiliser un lien. On peut faire un lien d'une page a.html vers une page b.html, mais on peut aussi faire un lien vers un autre site. Pour faire un lien, il faut utiliser la balise avec l'attribut href afin d'indiquer vers quelle page le lien doit conduire, ce qui donne par exemple : OpenClassrooms Pour placer le lien dans un paragraphe, on fait comme suit :

Bonjour. Vous souhaitez visiter OpenClassrooms ?
C'est un bon site ;o)

Par défaut, le lien s'affiche en bleu souligné. Et si on a déjà ouvert la page, le lien s'affiche en violet. CSS permettra de modifier ça. Pour faire un lien vers un autre site, il suffit donc de copier son URL. Si vous faites un lien vers un site qui comporte une adresse un peu bizarre avec des &, comme http://www.site.com/?data=15&name=mateo21 , vous devrez remplacer tous les « & » par « & » dans votre lien, comme ceci :http://www.site.com/?data=15&name=mateo21 . Vous ne verrez pas la différence, mais cela est nécessaire pour avoir une page web correctement construite en HTML5. Pour faire un lien renvoyant à une autre page de notre site, il faut d'abord créer deux fichiers correspondant à deux pages HTML différentes. Par exemple, page1.hmtl et page2.html. Deux fichiers qui doivent se trouver dans un même dossier. Il faudra utiliser la balise pour donner ce qu'on appelle un lien relatif (au contraire d'un lien absolu comme vu précédemment), ce qui donne par exemple :

Bonjour. Souhaitez-vous consulter la page 2 ?

Si les pages se trouvent dans un autre dossier, il faut faire apparaître le nom dudit dossier dans la balise. Par exemple, pour un dossier "contenu" : Lien vers page 3 Et s'il y avait eu d'autres dossiers, ça aurait pris cette forme : Ceci dit, si le fichier cible est placé dans un dossier plus haut dans l'arborescence, il faut écrire de cette manière avec deux points : L'exemple se trouve sur la page 3. bla Aller vers l'ancre J'ai placé une ancre au début pour la démonstration. Pour aller vers une ancre, il faut utiliser l'attribut href qui doit contenir un dièse suivi du nom de l'ancre. J'ai également placé un exemple en page 1. L'attribut id sert à donner un nom « unique » à une balise, pour s'en servir de repère. Et, croyez-moi, vous n'avez pas fini d'entendre parler de cet attribut. Ici, on s'en sert pour faire un lien vers une ancre mais, en CSS, il nous sera très utile pour « repérer » une balise précise, vous verrez. Évitez cependant de créer des id avec des espaces ou des caractères spéciaux ; utilisez simplement, dans la mesure du possible, des lettres et des chiffres pour que la valeur soit reconnue par tous les navigateurs. Il est aussi possible de faire un lien vers une ancre située dans une autre page. Il suffit de taper le nom de la page suivi d'un dièse suivi du nom de l'ancre. Exemple :
